WORDPRESS

W

Envie de créer un site web facilement ?
Utilisez Squarespace Wordpress !

WordPress est un système de gestion de contenu gratuit, libre et open-source. Ce logiciel écrit en PHP repose sur une base de données MySQL et est distribué par la fondation WordPress.org.

Sur ce service, vous pouvez trouver un site web fonctionnel, avec la possibilité de se connecter via un compte administateur vous donnant accès à toutes les options de personnalisations du site, que ce soit du contenu ou de l'apparence. Bref la classe.
